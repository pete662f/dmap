# World Reference Assets

This directory is generated by `../../scripts/prepare-style-assets.sh`.

It contains lightweight Natural Earth-derived GeoJSON assets used to render a low-resolution rest-of-world reference basemap beneath the Denmark vector tiles.

These files are served through TileServer's `/files/world-reference/` endpoint and referenced from the generated style via `file://world-reference/...` URLs so the runtime style keeps the global fallback layers intact.
